Eduplay is a project initially started by Yusuf Hassan. The intent is to produce a website which provides a number of educational games, which are both fun and supplemental to the user's education.
While Eduplay is created specifically for UK students in KS2 and KS3, I'm working to create games that are more vague, in order to appeal to the wider international community.
It's completely free, and will always remain this way! As the home page says, "Fun. Free. Forever."
