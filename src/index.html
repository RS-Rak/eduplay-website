About

Basic Info

Welcome to Eduplay. This is an educational website, focused on teaching younger students English, Maths, and Science through the use of minigames. If you'd like to access these minigames, alongside descriptions of what they do, please navigate to the games page below. By hovering over each icon, you can see more info on that game, and decide whether or not you'd like to play it. If you have any further questions, there are some FAQs at the bottom of this page.

Origins of Eduplay

Eduplay was created for my Extended Project Qualifications. Inspired by other great educational websites, such as Mathletics or Sheppard Software. The end goal is for there to be 9 minigames created, 3 each for Maths, Science and English. New games may be added in the future, as I become more comfortable with both the Phaser game engine and JavaScript as a whole.

FAQs

Eduplay is a project initially started by Yusuf Hassan. The intent is to produce a website which provides a number of educational games, which are both fun and supplemental to the user's education.
While Eduplay is created specifically for UK students in KS2 and KS3, I'm working to create games that are more vague, in order to appeal to the wider international community.
It's completely free, and will always remain this way! As the home page says, "Fun. Free. Forever."
